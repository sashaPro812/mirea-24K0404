# Практическое занятие 1
Программа представляет собой реализацию команду `cp` в Linux. Она копирует содержимое одного файла в другой файл. Используются системные вызовы `open()`, `read()`, `write()` и другие.

## Состав проекта
- `copy.c` - исходный код программы
- `Makefile` - файл для сборки проекта
- `original.txt` - тестовый исходный файл
- `copy.txt` - тестовый целевой файл (может создаться во время работы, если отсутствует в указанной директории)
- `Работа в терминале.odt` - отчёт со скриншотами мониторинга
- `copy` - исполняемый файл (после сборки)

## Сборка программы
```bash
make clean
make
```
где `make clean` используется для очистки предыдущей сборки

## Использование
`./copy <исходный файл> <целевой файл>`

Также требуется открыть два терминала. В первом терминале надо запустить программу, при этом не завершая её. В терминал выведется текущий PID процесса. Дальнейшая работа во втором терминале показана в файле`Работа в терминале.odt`.
